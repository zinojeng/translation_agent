## 貢獻指南

歡迎提交 Pull Request 或開設 Issue 來改進專案。在提交之前，請確保：

1. 程式碼遵循 PEP 8 規範
2. 新功能包含適當的測試
3. 更新文件以反映變更
4. 在更新日誌中記錄重要改動
